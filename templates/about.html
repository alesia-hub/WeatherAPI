About MyWeather API

MyWeather API is a simple API provides weather information for various locations around the world.

URL format: http://127.0.0.1:5001/api/v1/weather/station/date

URL Example: http://127.0.0.1:5001/api/v1/weather/10/18102025

URL Example to get weather for the station: http://127.0.0.1:5001/api/v1/weather/10
